# Core — общее ядро игрока

Сюда постепенно переносится логика, которую используют несколько систем (бой, стадион, магазин, рейтинг).

- `parse-json.js` — разбор JSON-полей из SQLite
- `player-stats.js` — эффективные статы (тату + шмот + прокачка)

Планируется: `currency.js`, `reputation.js`, `constants.js`.

См. [docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md).
